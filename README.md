## Testing  
To test you need to add a .env file in the root directory.

Below is the .env template:

```
DB_CONNECTION=mysql
DB_HOST=your_host
DB_PORT=3306
DB_DATABASE=your_database
DB_USERNAME=your_user
DB_PASSWORD=your_password
```
